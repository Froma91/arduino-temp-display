# Arduino Temperature Monitor

This project demonstrates real-time temperature monitoring using an LM35 sensor connected to an Arduino.

Temperature data is transmitted through the serial port and displayed in a web interface built with React and TypeScript using the Web Serial API.

## Current Features

- Real-time temperature acquisition from an LM35 sensor
- USB serial communication with Arduino
- Live temperature display
- Web Serial API integration
- React + TypeScript frontend

## Technologies

- Arduino
- LM35 Temperature Sensor
- React
- TypeScript
- TanStack Start
- Web Serial API
